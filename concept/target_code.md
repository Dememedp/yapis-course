## Идентификатор
целевой код (eng: target code)


## Определение
Целевой код - код, семантически эквивалентный промежуточному коду исходной программы, который может быть исполнен на
целевой платформе.

[Источник.](https://wiki.livid.pp.ru/students/sp/lectures/g.html)


## Примеры
- Машинный код
- Байт-код
